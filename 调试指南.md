# 贾维斯语音助手 - 调试指南

## 🐛 常见问题与解决方案

### 1. 安装失败问题

#### 问题1：“无法安装应用”或“解析包错误”
**可能原因**：
- APK文件下载不完整
- 设备存储空间不足
- 系统版本不兼容
- 安全设置阻止安装

**解决方案**：
1. **重新下载APK文件**：
   - 确保下载过程中网络稳定
   - 检查下载的APK文件大小是否为8.3MB
   - 重新下载：https://github.com/JarvisAI-CN/minimal-voip-app/releases/download/v1.0/app-debug.apk

2. **检查设备存储空间**：
   - 确保设备至少有100MB可用空间
   - 删除不需要的文件或应用释放空间

3. **检查Android系统版本**：
   - 该应用需要Android 8.0 (API 26)或更高版本
   - 查看设备系统版本：设置 → 关于手机 → Android版本

4. **启用未知来源应用安装**：
   - 在设备上打开“设置”
   - 找到“安全”或“隐私”选项
   - 启用“允许安装未知来源应用”
   - 如果有“安装未知应用”选项，确保允许浏览器或文件管理器安装

#### 问题2：“应用未安装”或“安装被阻止”
**解决方案**：
1. **清除应用缓存**：
   ```
   设置 → 存储 → 缓存数据 → 清除
   ```

2. **尝试使用ADB命令安装**：
   ```bash
   # 确保已安装ADB工具
   adb install -r app-debug.apk
   ```

### 2. 运行时问题

#### 问题1：应用崩溃或无法启动
**可能原因**：
- 设备系统资源不足
- 应用权限未正确授予
- 设备上有冲突的应用

**解决方案**：
1. **重启设备**：
   - 关闭并重新启动设备
   - 释放内存和系统资源

2. **检查应用权限**：
   - 确保已授予所需权限：
     - 麦克风权限
     - 网络权限
     - 存储权限

3. **卸载并重新安装**：
   - 完全卸载应用：设置 → 应用程序 → 贾维斯语音助手 → 卸载
   - 重新下载并安装

#### 问题2：语音识别功能不工作
**解决方案**：
1. **检查麦克风权限**：
   - 确保应用已允许使用麦克风
   - 设置 → 应用程序 → 贾维斯语音助手 → 权限 → 麦克风

2. **检查网络连接**：
   - 语音识别功能需要网络连接
   - 确保Wi-Fi或移动数据正常

3. **检查应用语言设置**：
   - 目前只支持中文识别
   - 确保设备语言设置为中文

### 3. 网络问题

#### 问题：应用无法连接到服务器
**解决方案**：
1. **检查网络连接**：
   - 确认Wi-Fi或移动数据连接正常
   - 尝试切换网络类型

2. **检查防火墙设置**：
   - 确保没有防火墙阻止应用的网络请求

3. **检查DNS设置**：
   - 尝试使用公共DNS服务器
   - 例如：8.8.8.8 或 114.114.114.114

### 4. 语音识别问题

#### 问题：无法识别语音
**解决方案**：
1. **确保语音清晰**：
   - 在安静的环境中说话
   - 保持适当的距离（10-30厘米）

2. **检查麦克风灵敏度**：
   - 检查设备麦克风是否正常工作
   - 尝试使用其他应用测试麦克风

3. **调整语音输入**：
   - 确保输入的是中文普通话
   - 说话速度适中，吐字清晰

### 5. 数据存储问题

#### 问题：无法保存任务或文件
**解决方案**：
1. **检查存储权限**：
   - 确保应用已允许使用存储权限

2. **检查存储空间**：
   - 确保设备有足够的可用空间

3. **清除应用数据**：
   ```
   设置 → 应用程序 → 贾维斯语音助手 → 存储 → 清除数据
   ```

**获取设备信息**：
```bash

```

### 📱 设备兼容性检查表

#### 系统要求
- ✅ Android 8.0 (API 26) 或更高
- ✅ 至少 2GB RAM
- ✅ 至少 100MB 可用存储空间
- ✅ 1GHz 或更快的处理器

#### 网络要求
- ✅ Wi-Fi 5GHz 或 2.4GHz
- ✅ 移动网络：4G 或 5G
- ✅ 网络延迟 ≤ 200ms
- ✅ 下载速度 ≥ 1Mbps

#### 硬件要求
- ✅ 内置麦克风
- ✅ 扬声器
- ✅ 触摸屏幕
- ✅ 存储设备（内部存储或SD卡）
